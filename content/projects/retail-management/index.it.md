## Panoramica

Un'applicazione web completa per la gestione retail, costruita per una boutique di moda, che gestisce tutti gli aspetti delle operazioni quotidiane. Completamente responsive e in produzione.

## Funzionalità

- {{< icon "list" >}} **Gestione anagrafiche** clienti e registro clienti
- {{< icon "tag" >}} **Inventario** e gestione magazzino
- {{< icon "code" >}} **Punto vendita** con integrazione hardware (stampanti scontrini, cassetti portavaluta)
- {{< icon "list-check" >}} **Tracciamento ore dipendenti** con report dettagliati
- {{< icon "heart" >}} **Programmi fedeltà** con carte regalo e livelli di sconto
- {{< icon "fire" >}} **Analisi vendite** con dashboard personalizzabili

## Strumenti utilizzati

PHP, JavaScript, Node.js, MySQL
