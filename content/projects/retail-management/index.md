## Overview

A complete retail management web application built for a fashion boutique, handling all aspects of day-to-day operations. Fully responsive and deployed in production.

## Features

- {{< icon "list" >}} **Client records** management and customer registry
- {{< icon "tag" >}} **Inventory** and warehouse management
- {{< icon "code" >}} **Point-of-sale** with hardware integration (receipt printers, cash drawers)
- {{< icon "list-check" >}} **Employee hour tracking** with detailed reports
- {{< icon "heart" >}} **Loyalty programs** with gift cards and discount tiers
- {{< icon "fire" >}} **Sales analytics** with customizable dashboards

## Tools Used

PHP, JavaScript, Node.js, MySQL
